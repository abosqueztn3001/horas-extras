🔥 Configurar Firebase (base de datos en tiempo real)

Firebase es gratuito. Con esta configuración todos los usuarios comparten la misma data en tiempo real desde cualquier dispositivo.

Paso 1 — Crear proyecto Firebase (gratis):
1. Ve a console.firebase.google.com
2. Clic en Agregar proyecto → nombre: horas-extras → Crear
3. En el menú izquierdo: Realtime Database → Crear base de datos
4. Selecciona región (ej: us-central1) → modo Prueba (30 días gratis, luego configura reglas)
5. En el menú izquierdo: Configuración del proyecto (ícono engranaje)
6. Desplázate a Tus apps → clic en </> Web → registra la app
7. Copia los valores de firebaseConfig y pégalos abajo

⚠️ Solo necesitas apiKey, authDomain, databaseURL y projectId. Los demás campos son opcionales.

⚙ Configurar Google Sheets

Cómo obtener la URL:
1. Abre Google Sheets → Extensiones → Apps Script
2. Pega el código del archivo apps_script.gs y guarda (Ctrl+S)
3. Implementar → Nueva implementación
4. Tipo: Aplicación web · Acceso: Cualquier persona
5. Copia la URL generada y pégala arriba
